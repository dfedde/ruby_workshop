for i in 1..5
  puts "My current favorite number is #{i}"
end
but that kinda sux and is not the ruby way
we could do this 

(1..5).each do |i|
  puts "My current favorite number is #{i}"
end
or even this 

5.times do |i|
  puts "My current favorite number is #{1 + i}"
